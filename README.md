📰 News Aggregator CLI
A simple and efficient Command Line Interface (CLI) application built with Python that fetches and displays the latest news from multiple sources using a News API. Stay informed directly from your terminal without opening a browser.
✨ Features
📰 Fetch the latest news headlines
📂 Browse news by category
🌍 Select news by country (optional)
🔍 Search news by keyword
⚡ Fast and lightweight CLI interface
🖥️ Clean and easy-to-read terminal output
🛠️ Technologies Used
Python 3
Requests Library
News API
Command Line Interface (CLI)

News-Aggregator-CLI/

🚀 Installation
Clone the repository:
git clone https://github.com/your-username/news-aggregator-cli.git
Navigate to the project folder:
cd news-aggregator-cli
Install the required dependencies:
pip install -r requirements.txt
🔑 API Key Setup
Get a free API key from NewsAPI.
Open config.py.
Add your API key:
API_KEY = "YOUR_API_KEY"
▶️ Run the Application
python main.py
